# 📌 Habit Tracker App

## 📖 Project Overview
The Habit Tracker App is a web application built using the .NET Blazor framework that helps users develop and maintain positive daily habits. Users can create an account, log in securely, and track their habits over time through a simple and intuitive interface.

This application is designed to promote productivity and consistency by allowing users to monitor their progress and stay accountable to their goals.

---

## 🎯 Project Objectives
- Build a fully functional web application using the .NET ecosystem
- Implement user authentication and authorization
- Perform full CRUD (Create, Read, Update, Delete) operations
- Apply best practices in UI/UX design, accessibility, and performance
- Collaborate effectively as a development team using GitHub and Trello
